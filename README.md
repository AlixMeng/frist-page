# Alix Go - 个人服务器导航页

> 一个现代化的个人服务器仪表盘，支持多模块展示、主题定制和管理功能。

![GitHub stars](https://img.shields.io/github/stars/alix/go?style=flat)
![GitHub license](https://img.shields.io/github/license/alix/go)

## 特性

### 核心功能
- 💬 **每日一言** - 名言轮播展示，支持自定义名言库
- <img width="1888" height="820" alt="image" src="https://github.com/user-attachments/assets/4debd6bc-167b-4552-88dc-bce6313ec9fd" />
- 📷 **摄影作品** - 图片画廊，支持多分类展示
- 📱 **应用导航** - 可自定义的应用图标网格
- 🖥️ **主机导航** - 服务器状态监控，绿/灰灯显示在线状态
- ⭐ **底部导航** - 常用网址快捷入口

### 管理功能
- 🔐 **用户认证** - 登录系统，支持管理员/普通用户
- 🎨 **主题样式** - 6套预设主题 + 自定义颜色/字体/圆角
- 🖼️ **照片管理** - 上传/删除照片，分类管理
- 🎵 **音乐管理** - 上传音乐文件，播放列表
- 📝 **每日一言管理** - 从文本文件批量导入
- 🖥️ **主机管理** - 添加/编辑/删除服务器
- 👁️ **显示设置** - 控制各模块显示/隐藏
- 🔒 **页面密码保护** - 访问密码验证

### UI/UX 特点
- 🌙 **深色主题** - 优雅的玻璃拟态设计
- ✨ **动画效果** - 流畅的过渡和悬停效果
- 📱 **响应式设计** - 支持桌面/平板/手机
- ♿ **无障碍支持** - 键盘导航、屏幕阅读器兼容
- 🎯 **即时生效** - 所有设置自动保存，无需手动保存

## 技术栈

- **前端**: HTML5, CSS3, Vanilla JavaScript
- **图标**: Font Awesome 6.4 (本地化) + 自定义 SVG 图标
- **存储**: localStorage (无后端依赖)
- **部署**: 任意静态文件服务器

## 快速开始

### 1. 克隆项目
```bash
git clone https://github.com/你的用户名/alix-go.git
cd alix-go
```

### 2. 启动服务
```bash
# Python
python3 -m http.server 8080

# 或 Node.js
npx serve .

# 或 PHP
php -S localhost:8080
```

### 3. 访问
浏览器打开 http://localhost:8080

## 功能说明

### 每日一言
- 支持从 TXT 文件导入（格式：句子 + 空格 + 作者）
- 8秒自动轮播
- 点击立即切换

### 摄影作品
- 4个分类：风景、人像、人文、其他
- 10秒自动刷新
- 支持上传自己的照片（Base64 存储）

### 主机导航
- 添加服务器名称、URL、图标
- 自动检测在线状态（通过 favicon 加载）
- 绿点 = 在线，灰点 = 离线

### 主题样式
**预设主题**:
- 默认主题（蓝黑）
- 深邃黑
- 海洋蓝
- 森林绿
- 日落橙
- 梦幻紫

**自定义选项**:
- 主色调
- 背景色
- 字体选择
- 圆角大小
